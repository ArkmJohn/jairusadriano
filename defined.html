Defined Booklet

Defined is a personal project of mine. It is a booklet I created compiled with different images that show beautiful meanings.

Overview

This project was created at the end of 2018. Defined was inspired by different places in the Philippines that had so much character that it had its own perfect definition for every location. This booklet is a full layout of the photos that I saw fit to have its own shining moment.
